# Taskoria ⚔️

**Taskoria** — это геймифицированный таск-менеджер с RPG-элементами.  
Проект про то, чтобы превращать повседневные задачи в прокачку персонажа и маленькую игру жизни.

---

## ✨ Идея

Обычные задачи часто ощущаются скучно и тяжело.  
Taskoria делает из них систему прогресса:

- задачи = квесты  
- выполнение = опыт  
- система прогресса = развитие персонажа  
- магазин/бонусы = "заклинания" для усиления продуктивности  

---

## 🔥 Функционал (MVP)

- Создание / удаление задач
- Отметка выполнения задач
- Простая система прогресса (XP / уровень)
- Базовая геймификация
- Хранение данных через Supabase

---

## 🧱 Стек технологий

- React / Vite
- TypeScript
- Supabase (backend + auth + database)
- TailwindCSS / UI компоненты
- React Router

---

## 🗂 Структура проекта (пример)
